# Project Salus — Marketing Language

Phrases and terminology that resonate with defence/security customers and proposal reviewers.

---

## Capability Descriptors

- **Adversarial Path Planning** — autonomous discovery of worst-case threat approach routes exploiting terrain and sensor gaps
- **Terrain-Aware Coverage Analysis** — sensor viewsheds computed against real Digital Elevation Models, not flat-plane assumptions
- **Physics-Based Detection Modelling** — line-of-sight, slant range, azimuth, and elevation gating derived from sensor specifications
- **3D Engagement Geometry** — full volumetric detection modelling across altitude bands, not 2D ground-plane approximations
- **Sensor Network Redundancy Analysis** — per-cell coverage depth mapping; quantified degraded-mode performance under sensor loss
- **Worst-Case Corridor Identification** — systematic bearing sweep to surface maximum detection-gap approach vectors
- **Sub-Second Detection Timeline** — binary search refinement to sub-metre / sub-second first-detection precision
- **Kill Chain Integration** — detection-to-engagement handoff timing against sensor and effector coverage geometry
- **Engagement Calculation** — high-fidelity replay of a specific named threat trajectory through the sensor network

## Proposal / Tender Language

- **Evidence-Based Site Coverage Certification** — quantified coverage metrics suitable for inclusion in capability statements
- **Scenario-Driven Vulnerability Assessment** — structured simulation of threat profiles against real site geometry
- **Sensor Placement Optimisation** — data-driven sensor positioning to close identified coverage gaps
- **Multi-Threat Profile Analysis** — simultaneous evaluation across diverse UAS threat classes (commercial, FPV, fixed-wing)
- **Degraded Mode Analysis** — coverage modelling under partial sensor failure or jamming conditions

## Technical Differentiators (for tender responses)

- Site-specific terrain ingestion (GeoTIFF DEM/DSM)
- Sensor-model fidelity: per-platform azimuth arc, elevation arc, boresight, range bands, mounting height
- Piecewise-linear trajectory approximation with configurable fidelity
- Automated worst-case discovery — no analyst intuition required to find gaps
