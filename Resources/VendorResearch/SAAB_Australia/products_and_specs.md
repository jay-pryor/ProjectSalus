## Giraffe 1X Radar

### Overview
Giraffe 1X is a compact, man-portable 3D Active Electronically Scanned Array (AESA) radar for short-range air surveillance, GBAD target acquisition, and counter-RAM sense and warn. It uses the latest radar technology including Gallium Nitride (GaN) circuits.

### Key Specifications

| Parameter | Value |
|---|---|
| **Radar Type** | 3D AESA (Active Electronically Scanned Array) |
| **Frequency Band** | X-band (8-12 GHz) / NATO I-band |
| **Instrumented Range** | 75 km |
| **Maximum Detection Range** | ~180 km (for large aircraft); shorter for small UAS/low-RCS targets |
| **Coverage** | 360-degree azimuth |
| **Update Rate** | Full search volume every 1 second |
| **Total System Weight** | Less than 150 kg |
| **Topside Unit Weight** | ~100 kg |
| **Semiconductor Technology** | Gallium Nitride (GaN) |
| **Transport** | Pickup truck-sized vehicle, helicopter, or towed trailer |
| **Deployment** | Mobile, deployable, or fixed asset |
| **Operation** | Remote or local |
| **Power Requirements** | Low (details not publicly specified) |

### Detection Capabilities
- Fixed and rotary wing aircraft
- Fast missiles and RAM targets
- Small UAVs in high-clutter environments
- Simultaneous air surveillance, GBAD target acquisition, and C-RAM sense and warn without performance degradation

### Key Orders and Operators
- UK Ministry of Defence: 11 Giraffe 1X radars (May 2023)
- Sweden FMV: Giraffe 1X for ground-based air defence (August 2024)
- US Army: Contract for Giraffe 1X radars (~USD 46 million, Q3 2025)
- Latvia: procurement ongoing

## Giraffe 4A Radar

### Overview
Giraffe 4A is a truly multifunctional long-range 3D AESA radar combining air surveillance, weapon location, and ground-based air defence in a single system. It merges battle-proven designs from the Arthur weapon-locating radar and Giraffe AMB families with an all-new AESA sensor using GaN components.

### Key Specifications

| Parameter | Value |
|---|---|
| **Radar Type** | 3D AESA with digital beam forming |
| **Frequency Band** | S-band |
| **Instrumented Range (Land)** | Up to 400 km |
| **Instrumented Range (Sea Giraffe 4A)** | 350 km |
| **Elevation Coverage** | Greater than 70 degrees |
| **Rotation Rate** | 30 or 60 rpm |
| **Search Mode** | 360-degree or sector mode |
| **Sector Mode Update Rate** | Up to 8 Hz |
| **Target Revisit Time** | 1 second |
| **Track Capacity (Sea variant)** | Air: >1,000 tracks; Surface: >500 tracks |
| **Power Consumption (Sea variant)** | 40 kW |
| **Semiconductor Technology** | Gallium Nitride (GaN) |
| **Deployment Time** | Less than 10 minutes (2 personnel) |
| **Transport** | Airliftable in a single C-130 load |
| **Data Interface** | Ethernet |
| **Swedish Designation** | PM24 |

### Roles
- **Air Surveillance**: 360-degree coverage against all air targets from UAVs to supersonic missiles
- **Weapon Locating**: Tracks ballistic projectiles; classifies and calculates point of origin and point of impact for counter-battery fire
- **GBAD**: Provides engagement-quality target data for VSHORAD to MRAD effectors

### Counter-UAS Capability
- Detects low, slow, and small targets including UAVs
- Enhanced Low, Slow and Small (ELSS) function (developed on Giraffe AMB, applicable across Giraffe family) detects targets with RCS as small as 0.001 sq metres

### Key Operators
- Colombia (1), Finland (4), Germany (4), Sweden, Thailand (1), United States (11+ units delivered or on order)

## Giraffe AMB Radar

### Overview
Giraffe AMB (Agile Multi-Beam) is a medium-range 3D radar surveillance system serving as the primary sensor for integrated medium- and short-range GBAD. It is combat-proven with over 150,000 hours of full operation.

### Key Specifications

| Parameter | Value |
|---|---|
| **Radar Type** | 3D agile multi-beam, stacked beam with monopulse elevation |
| **Frequency Band** | C-band (G/H-band) |
| **Frequency (Sea Giraffe AMB)** | 5.4-5.85 GHz |
| **Output Power (Sea variant)** | 25 kW |
| **Receiver Beams** | 12 simultaneous receiver beams in elevation (Sea variant) |
| **Rotation Rate** | 60 and 30 rpm |
| **Elevation Coverage** | 0-70 degrees |
| **Target Update Rate** | 1 second |
| **Antenna Height** | 12 m above ground (hydraulically operated mast) |
| **Housing** | Fully self-contained 20ft ISO container cabin |
| **Deployment Time** | Less than 10 minutes |
| **Teardown Time** | Less than 5 minutes |
| **Operation** | Remote or local; networked air picture |
| **Combat Record** | 24/7 capability; 150,000+ hours in full operation |

### Detection Capabilities
- Fixed wing aircraft and supersonic fighters
- Cruise missiles
- Helicopters
- Small, slow, and low-flying UAVs
- Incoming rockets, artillery, and mortar rounds (simultaneous RAM warning)
- Jammer strobes

### Enhanced Low, Slow and Small (ELSS) Function
- Automatically finds and classifies low, slow, and small targets
- Detects targets as slow and small as quadcopter UAS
- Demonstrated detection of targets with RCS as small as 0.001 sq metres
- Operates simultaneously with full suite of regular air surveillance functions
- Successfully tested during Bristow 15 trials against complex tactical drone profiles

### Key Operators
- Estonia: 5 mobile truck-mounted units (Air Defence Battalion)
- United Kingdom: Jointly operated by British Army and RAF (49 (Inkerman) Battery Royal Artillery)
- United States: Sea Giraffe AMB on Independence-class LCS (designated AN/SPS-77(V)1 and V(2))
- Sweden: Army and Navy (including Visby-class corvettes)
- Algeria: MEKO A-200 frigates
- Canada: Halifax-class frigates

## RBS 70 / RBS 70 NG VSHORAD

### Overview
RBS 70 NG (New Generation) is a very short range air defence (VSHORAD) system featuring a laser beam-riding missile. Introduced in 2011, it is an upgraded version of the RBS 70 system operational with 19 countries. The laser beam-riding guidance makes it highly resistant to jamming and countermeasures.

### Key Specifications -- RBS 70 NG

| Parameter | Value |
|---|---|
| **System Type** | VSHORAD (Very Short Range Air Defence) |
| **Guidance** | Laser beam-riding |
| **Engagement Range** | 9,000 m (9 km) |
| **Height Coverage** | 5,000 m |
| **Missile Length** | 132 cm (without booster) |
| **Missile Diameter** | 10.6 cm |
| **Fin Span** | 32 cm |
| **Missile Weight** | 15 kg |
| **Maximum Velocity (BOLIDE)** | Mach 2 |
| **Warhead** | Combined shaped-charge and pre-fragmented |
| **Operators Required** | 1 (operation); 3 (portability) |
| **Sight** | NG sight with integrated high-resolution thermal imager |
| **Components** | Launch container, tripod, and NG sight |

### Earlier RBS 70 Variants (Mk 1/Mk 2)

| Parameter | Value |
|---|---|
| **Engagement Range** | 5,000-6,000 m |
| **Ceiling** | 3,000 m |

### Missile Variants
- **Mk 0**: Original variant
- **Mk 1**: Improved range
- **Mk 2**: Further improvements
- **BOLIDE (4th Generation)**: All-target VSHORAD missile; Mach 2 speed; combined shaped-charge and pre-fragmented warhead

### Warhead Capability
- Defeats armoured aerial targets (attack helicopters, CAS aircraft)
- Effective against ground-based armoured targets (APCs)
- Counter-UAS capable

### Modularity
- Compatible with all existing generations of RBS 70 missiles
- NG sight modular design allows integration into virtually any vehicle platform
- Can be integrated into networks and remotely controlled platforms

### Operators (18+ countries)
Australia, Argentina, Bahrain, Brazil, Czech Republic, Finland, Germany, Indonesia, Iran, Ireland, Latvia, Norway, Pakistan, Singapore, Sweden, Thailand, Tunisia, UAE

## MSHORAD Integrated Solution

### Overview
Saab's Mobile Short-Range Air Defence (MSHORAD) solution combines the Giraffe 1X radar, C2 command and control platform, and RBS 70 NG Remote Weapon Station (RWS) into a unified mobile air defence system.

### Components

| Component | Role |
|---|---|
| **Giraffe 1X** | Air surveillance radar and target acquisition |
| **C2 Platform** | Command, control, and communications |
| **RBS 70 NG RWS** | Remote Weapon Station with BOLIDE missiles |

### Key Features
- Each component is small, lightweight, and man-portable
- Can be installed on any vehicle platform
- Modular design allows installation alongside existing equipment
- Specifically engineered to identify and neutralize small-to-medium UAS flying low with minimal signatures

### Key Orders
- Lithuania: 3 orders for MSHORAD system; latest valued at ~SEK 1.4 billion (deliveries 2026-2030)
- Sweden (FMV): Sensors and C2 systems for brigade-level GBAD solution; ~SEK 2.1 billion (deliveries 2027-2028)

## Nimbrix Counter-UAS Missile

### Overview
Nimbrix is Saab's first dedicated Counter-UAS missile, revealed in 2025. It is a fire-and-forget missile specifically developed to counter the increasing threat from small drones on the battlefield.

### Key Specifications

| Parameter | Value |
|---|---|
| **Type** | Dedicated C-UAS missile |
| **Guidance** | Fire-and-forget with active seeker |
| **Engagement Range** | Up to 5 km |
| **Warhead** | Hard-kill warhead with air-burst mode for UAS swarm engagement |
| **Footprint** | Small |
| **Cost** | Low cost (designed for affordability against drone threats) |
| **Mounting** | Vehicles or fixed configurations; 3 or 4 missiles per side on M-SHORAD Trackfire turret |
| **First Deliveries** | Targeted for 2026 |

## LOKE Counter-Drone Concept

### Overview
LOKE (Light Operational Kinetic Electronic) is a Swedish C-UAS concept developed in only 84 days by FMV and Saab, field-tested in June 2025 in the Gotland region.

### Components

| Component | Role |
|---|---|
| **Giraffe 1X** | Radar sensor |
| **Lightweight C2 System** | Command and control |
| **SHORAD Effectors** | Kinetic defeat |
| **Third-party Subsystems** | Additional integrated capabilities |
