## Coyote Block 2+ Interceptor

### Overview
The Coyote Block 2+ is a low-cost, rail-launched counter-UAS interceptor missile with a boost rocket motor and turbine engine. It functions as a radar-guided surface-to-air missile optimized for defeating Group 1-3 UAS targets. It has been combat-proven with 170+ successful interceptions across deployments in the Middle East, Africa, and Europe.

### Key Specifications

| Parameter | Value |
|---|---|
| **Type** | Expendable C-UAS interceptor (surface-to-air guided missile) |
| **Propulsion** | Rocket booster + turbine sustainment motor |
| **Engagement Range** | 10-15 km (6.2-9.3 mi) |
| **Speed** | 555-595 km/h (345-370 mph) |
| **Flight Time** | Up to ~4 minutes |
| **Weight (Block 1 reference)** | ~13 lb (5.9 kg); Block 2 believed similar |
| **Warhead** | Tungsten fragmentation warhead; forward-firing, low collateral blast fragmentation |
| **Guidance** | Datalink (C-band) + onboard seeker + proximity fuze for terminal phase |
| **Fire Control** | FAAD Command and Control system |
| **Target Set** | Group 1-3 UAS |
| **Re-attack Capability** | Yes -- can re-engage target if first pass misses |
| **Unit Cost** | ~$100,000 per round |

### Platform Integration
- Oshkosh JLTV 4x4 vehicle
- M-ATV 4x4 armored vehicle (M-LIDS configuration)
- Fixed-site installations (FS-LIDS)
- US Navy Arleigh Burke-class destroyers (planned)

### Combat Record
- 170+ confirmed kills in operational deployments
- Deployed at 36+ sites outside the United States
- Successfully demonstrated against 30+ target swarms

### Procurement
- US Army: 6,000 Block 2 interceptors planned (2025-2029) at ~$100,000 each
- 700 Block 3 (non-kinetic) variants planned
- $75 million contract for 600 Coyote 2C Interceptors (January 2024)

## Coyote Block 3NK (Non-Kinetic)

### Overview
The Coyote Block 3NK is a non-kinetic counter-UAS variant that defeats drone swarms using an electronic warfare payload, minimizing collateral damage. It can be recalled and redeployed for additional engagements.

### Key Specifications

| **Reusability** | Recallable and redeployable for multiple engagements |

| **Procurement** | 700 units planned (2025-2029) |

## KuRFS (Ku-band Radio Frequency Sensor)

### Overview
KuRFS is a 360-degree AESA radar providing persistent detection, identification, and tracking of airborne threats from rockets, artillery, and mortars to all classes of drones. Originally built for C-RAM missions in Iraq and Afghanistan, it has proven highly effective for counter-UAS operations.

### Key Specifications

| Parameter | Value |
|---|---|
| **Radar Type** | AESA (Active Electronically Scanned Array) |
| **Frequency Band** | Ku-band (12-18 GHz range) |
| **Coverage** | 360 degrees, persistent |
| **Detection Range (Class I UAS)** | 16 km (9.9 mi) |
| **Minimum Detectable Object** | Objects as small as a 9 mm bullet |
| **Setup Time** | Within 30 minutes |
| **Deployment** | Fixed location or vehicle-mounted |
| **Swarm Tracking** | Successfully detected and tracked 30+ target swarms |
| **Integrated Weapons** | 15+ weapon systems including Coyote, Phalanx, Phaser HPM, HELWS |
| **Weight** | Not publicly disclosed (likely classified) |
| **Exact Frequency** | Not publicly disclosed (export-controlled) |

### Key Features
- Ku-band short wavelengths produce significantly sharper image resolution than other bands
- Can distinguish between many small objects simultaneously (bird vs. drone discrimination)
- Provides fire control quality data for Coyote effector engagement
- Core sensor for LIDS (both M-LIDS and FS-LIDS configurations)

## NASAMS

**National Advanced Surface-to-Air Missile System**

### Overview
NASAMS is a medium-range, network-centric air defence system jointly developed by Raytheon and Kongsberg Defence & Aerospace. It can engage aircraft, helicopters, cruise missiles, and UAVs, and is the world's first operational network-centric ground-based air defence system. Delivered or under production for 15 nations.

### Key Specifications

| Parameter | Value |
|---|---|
| **Type** | Medium-range ground-based air defence system |
| **Maximum Range** | 33 km |
| **Maximum Altitude** | 15 km (50,000 ft) |
| **Simultaneous Engagements** | 72 targets (active and passive modes) |
| **Primary Missile** | AIM-120 AMRAAM |
| **Extended Range Missile** | AMRAAM-ER (significantly extends engagement volume) |
| **Short Range Missile** | AIM-9X-2 Sidewinder |
| **Surveillance Radar** | Raytheon MPQ-64F1 Sentinel (3D pencil beam) |
| **EO/IR Sensor** | Raytheon AN/AAS-52 MTS-A (multispectral targeting system with laser rangefinder) |
| **C2 System** | Kongsberg Fire Distribution Center (FDC) |

### Counter-UAS Integration
- NASAMS can integrate with HELWS for drone swarm defeat (demonstrated March 2022)
- Compatible with future active/passive radars and sensors
- Sense and Warn capability
- Integration with C-RAM, C-UAS, and Raytheon Family of Missiles
- GhostEye MR radar upgrade expands detection range and altitude coverage

### Notable Deployments
- Washington, D.C. airspace protection (White House area)
- Ukraine (provided by the United States)
- 15 nations worldwide

## Stinger (FIM-92)

### Overview
The FIM-92 Stinger is a man-portable air-defense system (MANPADS) using infrared homing. In service since 1981 with 30 countries, it has been adapted for counter-drone missions with proximity fuze upgrades. Manufactured by Raytheon, with licensed production by Airbus Defence and Space (Germany) and Roketsan (Turkey).

### Key Specifications

| Parameter | Value |
|---|---|
| **Type** | MANPADS (Man-Portable Air-Defense System) |
| **Guidance** | Infrared homing (proportional navigation + airframe tracking) |
| **Missile Length** | 5.0 ft (1.52 m) |
| **Missile Diameter** | 2.8 in (70 mm); fins 3.9 in (100 mm) |
| **Missile Weight** | 22 lb (10.1 kg) |
| **System Weight** | ~34 lb (15.2 kg) including launcher, gripstock, and IFF |
| **Targeting Range** | Up to 4,800 m (15,700 ft) |
| **Engagement Altitude** | Up to 3,800 m (12,500 ft) |
| **Kinematic Range** | ~8,000 m (26,000 ft) |
| **Maximum Speed** | Mach 2.54 (1,930 mph / 864 m/s) |
| **Warhead** | 3 kg penetrating hit-to-kill warhead with impact fuze and self-destruct timer |
| **Propulsion** | Two-stage solid-fuel sustainer |

### Key Variants

| Variant | Description |
|---|---|
| **FIM-92A** | Initial production model |
| **FIM-92B (POST)** | Dual-band IR/UV seeker with rosette-pattern scanning |
| **FIM-92C (RMP)** | Enhanced resistance to countermeasures; reprogrammable microprocessor |
| **FIM-92E (RMP Block I)** | Improved flight behavior against small targets (drones, cruise missiles) |
| **FIM-92J** | Block 1 upgrade with proximity fuze for enhanced C-UAS capability; 10-year life extension |

### Counter-Drone Adaptations
- Proximity fuze retrofit enables destruction of UAS by detonating warhead near the target
- FIM-92J specifically designed for counter-drone effectiveness
- Successfully destroyed drone targets during Army tests (2017)
- Compatible with Javelin LWCLU (Lightweight Command Launch Unit) for ground-based drone defense

### Future
- US Army seeking MANPADS replacement (request for information issued 2020)
- RTX and Lockheed Martin competitively developing Stinger replacement (selected September 2023)

## LIDS (Integrated Defeat System)

**Low, Slow, Small Unmanned Aircraft Integrated Defeat System**

### Overview
LIDS is the US Army's primary deployed counter-UAS solution, integrating Raytheon's KuRFS radar and Coyote effectors with Northrop Grumman's FAADC2 and SRC's electronic warfare system.

### Variants

#### FS-LIDS (Fixed Site)

| Parameter | Value |
|---|---|
| **Type** | Fixed-site counter-drone system |
| **Components** | KuRFS radar, EO/IR sensors, EW system, Coyote launchers |
| **Detection** | Radar, cameras, electronic warfare |
| **Defeat** | Coyote interceptors + EW |
| **First International Customer** | Qatar ($1 billion defense agreement, May 2025) |

#### M-LIDS (Mobile)

| Parameter | Value |
|---|---|
| **Type** | Mobile counter-drone system |
| **Platform** | Two M-ATV 4x4 armored vehicles |
| **Vehicle 1** | Turret with EO/IR sensors, 2-round Coyote launcher, 30 mm XM914 cannon, mast-mounted KuRFS radar |
| **Vehicle 2** | Additional sensors and EW capabilities |
| **Fire Control** | FAAD Command and Control |

### Procurement
- US Army: 6,700 Coyote interceptors planned for M-LIDS and FS-LIDS (2025-2029)

## Phaser High-Power Microwave

### Overview
Phaser is a containerized high-power microwave (HPM) directed-energy weapon that uses electromagnetic radiation to disable drones -- individually or in swarms -- at the speed of light. It was the first directed-energy weapon ever fielded by the US Air Force.

### Key Specifications

| Parameter | Value |
|---|---|
| **Type** | High-power microwave directed-energy weapon |
| **Form Factor** | Containerized on 20-foot trailer |
| **Power Source** | Internal diesel generator |
| **Beam Type** | Wide, conical microwave energy beam |
| **Technology** | Vacuum tube microwave generation with reflector antenna |
| **Targeting Sensors** | Radar and EO/IR cameras for cueing |
| **Modes** | "Disrupt" mode and "Destroy" mode |
| **Anti-Swarm Demonstrated** | 33 drones downed (2-3 at a time) in a single exercise |
| **Engagement Speed** | Milliseconds (faster than laser weapons) |
| **Cost Per Shot** | Cents per firing |
| **Magazine** | Unlimited |
| **Effective Range** | Classified |
| **Power Output** | Classified |

### Key Advantages
- Speed: millisecond engagement time vs. seconds for lasers
- Anti-swarm: defeats multiple drones simultaneously
- Unlimited magazine for sustained attacks
- Extremely low cost per engagement
- Can be paired with HELWS for complementary defense

## GhostEye Radar Family

### Overview
GhostEye is Raytheon's next-generation family of AESA radars using military-grade GaN technology. The family includes LTAMDS (for Patriot replacement) and GhostEye MR (for NASAMS integration). Relevant to counter-UAS as part of layered air defense.

### Key Specifications

| Parameter | Value |
|---|---|
| **Radar Type** | AESA (Active Electronically Scanned Array) |
| **Coverage** | 360 degrees |
| **Frequency Bands** | C-band primary; additional S-band and X-band |
| **Semiconductor Technology** | Gallium Nitride (GaN) |
| **Array Configuration (LTAMDS)** | 3 antenna arrays (1 primary front, 2 secondary rear) |
| **Primary Array Power** | More than 2x the power of current Patriot radar |
| **Missions** | IAMD, counter-UAS, critical infrastructure defense |
| **Threats Detected** | Cruise missiles, drones, fixed-wing, rotary-wing aircraft |
| **Weight** | Not publicly disclosed |
| **Detection Range** | Not publicly disclosed (classified) |

### Variants

| Variant | Role |
|---|---|
| **GhostEye / LTAMDS** | Lower Tier Air and Missile Defense Sensor; Patriot radar replacement |
| **GhostEye MR** | Medium-range variant for NASAMS integration; counters long-range mass attacks |

### Procurement
- US Army: $1.7 billion for LTAMDS low-rate initial production
