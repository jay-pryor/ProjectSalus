## VAMPIRE System Family

**Vehicle-Agnostic Modular Palletized ISR Rocket Equipment**

### Overview
VAMPIRE is an affordable, compact ISR and counter-unmanned weapons system designed to deploy on nearly any platform, vehicle, or vessel. First demonstrated as a prototype in 2021, it has been combat-tested in Ukraine where it has downed hundreds of hostile drones. L3Harris acquired a new production facility in Huntsville, Alabama, capable of producing 20-40 systems per month.

### Key Specifications

| Parameter | Value |
|---|---|
| **Munition** | BAE APKWS (Advanced Precision Kill Weapon System) -- laser-guided 2.75-inch (70mm) Hydra-70 rocket |
| **Engagement Range** | Up to 5 km |
| **Magazine Capacity** | 4 rounds (standard) / 12 rounds (Stalker XR variant) |
| **Sensor** | L3Harris WESCAM MX-10D RSTA targeting and EO/IR sensor |
| **Software** | Widow mission management software |
| **Guidance** | Laser-guided (semi-active laser homing) |
| **Fuze** | L3Harris proximity fuze |
| **Platform** | Vehicle-agnostic; fielded on Humvees, pickup trucks, and other light tactical vehicles |
| **Weight** | Not publicly disclosed; described as compact and rapidly deployable on light vehicles |
| **Cost** | Single-digit millions of dollars per unit |

### VAMPIRE Variants

| Variant | Description |
|---|---|
| **VAMPIRE (Baseline)** | 4-round APKWS launcher on light tactical vehicle with WESCAM MX-10D |
| **Stalker XR** | Upgraded land vehicle variant with 12-round launcher for increased firepower |
| **CASKET** (Containerized Anti-drone System with Kinetic Effects Turret) | Self-contained VAMPIRE-in-a-box for rapid deployment to remote locations |
| **BAT** (Base Anti-Drone Turret) | Fixed-site protection using automatic weapons and non-kinetic effects; minimizes collateral damage |
| **Black Wake** | Maritime variant mounted on an uncrewed surface vessel (USV / drone boat) |
| **Dead Wing** | Airborne variant (limited public details) |
| **Killcode** | Electronic warfare variant using jammers to disrupt drone threats (see EW section below) |

### Planned Upgrades
- Integration of radar for target search and track
- Integration of kinetic weapons (heavy and light machine guns)
- Integration of non-kinetic weapons (jammers and electronic attack)
- AI integration for extended detection range and improved operator reaction time

## Drone Guardian

### Overview
Drone Guardian is a sensor-agnostic and effector-agnostic counter-sUAS solution designed for strategic fixed-site protection. Built on an open, flexible architecture, it integrates best-in-breed sensors and effectors from multiple vendors.

### Key Specifications

| Parameter | Value |
|---|---|
| **Architecture** | Open, modular, configurable, and scalable |
| **Sensors** | Integrates radar, RF detection, and EO/IR sensors from multiple vendors |
| **RF Detection Frequency** | Wide frequency capability from 100 MHz to above 10 GHz (via CRFS RF sensors) |
| **Drone Detection Altitude** | Most integrated sensors detect drones below 100 m |
| **EW/Jamming** | CORVUS ICN EW solutions: jamming across 20 MHz to 6 GHz range |
| **Defeat Capabilities** | Non-kinetic (BROADSHIELD LCS, CORVUS ICN jamming) and kinetic effectors |
| **C2 Software** | Battle-proven command and control with multi-sensor data correlation |
| **Detection Range** | Dependent on integrated sensor suite (not publicly specified as a fixed value) |

### Key Features
- Multi-sensor, multi-layered approach to detection
- Real-time alerts and automated drone detection
- Reduces manpower requirements through automation
- Capable of engaging individual drones and drone swarms

## CORVUS-RAVEN

### Overview
CORVUS-RAVEN is a lightweight, highly portable counter-drone system providing passive signal detection, enhanced situational awareness, and electronic attack/jamming capability. It is designed for dismounted soldiers and vehicle-mounted operations.

### Key Specifications

| Parameter | Value |
|---|---|
| **Detection Method** | Passive RF signal detection |
| **Detection Range** | Up to 4 km (passive detect) |
| **Core Technology** | Individual CORVUS Node (ICN) -- multirole software-defined radio |
| **Jamming Range** | 20 MHz to 6 GHz (comms-denial and counter-drone) |
| **Missions Supported** | Counter-IED, electronic surveillance, electronic attack, C-sUAS |
| **Weight** | Not publicly disclosed; described as lightweight and wearable by individual soldiers |
| **Configurations** | Vehicle-mounted (e.g., Coyote vehicle) and dismounted/wearable |
| **Interoperability** | Networkable; ICNs provide holistic battlespace view |
| **Target Set** | Primarily Class 1 UAS systems |

### Key Features
- Passive detection with visual display of bearing to C2/battle management applications
- Networked ICN nodes provide greater visibility of small UAV threats
- Supports streamlined airspace management and GBAD integration
- Battle-tested: successfully neutralized hundreds of hostile drone attacks in combat

## VAMPIRE Killcode (EW Variant)

### Overview
The VAMPIRE Killcode is the dedicated electronic warfare variant of the VAMPIRE family. It uses jammers to disrupt and defeat drone threats non-kinetically.

### Key Specifications

| Parameter | Value |
|---|---|
| **Type** | Non-kinetic electronic warfare / jammer |
| **Platform** | Vehicle-mounted (VAMPIRE family platform) |
| **Defeat Method** | RF jamming to disrupt drone control links and navigation |
| **Details** | Limited public specifications; part of broader VAMPIRE family expansion announced in late 2025 |

## Red Wolf and Green Wolf

### Overview
The Wolf Pack launched effects platforms enable forces to neutralize hostile drones at the source, shifting from a defensive to an offensive posture.

### Key Specifications

| Parameter | Value |
|---|---|
| **Type** | Launched effects platforms |
| **Defeat Methods** | Both kinetic and electronic warfare options |
| **Platform Compatibility** | Fixed wing, rotary wing, UAV, ground, and maritime platforms |
| **Operational Concept** | Offensive counter-drone -- engage hostile drones at their source |
| **Details** | Limited public specifications available |

## BROADSHIELD LCS

### Overview
BROADSHIELD LCS is L3Harris's omni-directional non-kinetic electronic countermeasure system, integrated into the Drone Guardian solution for counter-sUAS jamming and disruption.

### Key Specifications

| Parameter | Value |
|---|---|
| **Type** | Omni-directional non-kinetic electronic countermeasure |
| **Function** | RF jamming / drone link disruption |
| **Integration** | Part of Drone Guardian and broader L3Harris C-UAS portfolio |
| **Details** | Limited public specifications; described as battle-proven ECM |
