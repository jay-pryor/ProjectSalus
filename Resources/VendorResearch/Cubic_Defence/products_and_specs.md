# Cubic Defence Australia - Products and Specifications

## Company Overview

Cubic Defence Australia is part of Cubic Corporation, a global provider of integrated systems and services for Defence training, secure communications, and transportation markets. Established in Australia in 2006 (Townsville), the company has grown to over 300 full-time and casual staff. Veterans make up approximately 60% of the workforce and 70% of the leadership team.

Cubic provides integrated Live, Virtual, Constructive (LVC) training and simulation solutions, as well as C4ISR secure communications and networking products to the Australian, New Zealand, US, and allied nations' Defence Forces.

**Quality Certification**: ISO 9001:2015

## Training and Simulation Systems

### I-MILES IWS (Individual Weapon System)

Man-wearable laser engagement training system used for force-on-force exercises.

| Parameter | Value |
|---|---|
| **System Type** | Laser-based tactical engagement simulation |
| **Components** | Laser emitters (weapon-mounted) + on-body laser sensors |
| **Function** | Simulates weapon fire using lasers and blank cartridges |
| **Casualty Assessment** | Real-time; uses random number roll and casualty probability lookup table |
| **Data Encoded in Laser** | Shooter identity, weapon type, ammunition type |
| **Range Simulation** | Each laser transmitter mimics the effective range of its host weapon |
| **Current US Version** | MILES IWS2 (as of 2012) |
| **Interoperability** | Compatible with TESS laser harnesses (Cubic/Oscmar, New Zealand); I-HITS harnesses |

### I-MILES TVS (Tactical Vehicle System)

Vehicle-mounted adaptation of the man-worn IWS.

| Parameter | Value |
|---|---|
| **System Type** | Vehicle-mounted laser engagement simulation |
| **Architecture** | Modular, open architecture (adaptable across Army vehicle fleet) |
| **Interface** | Touch-screen displays with intuitive graphics |
| **Communications** | Wireless |
| **Function** | Real-time casualty assessment for force-on-force vehicle training |
| **Design** | COTS technology for simplified setup, installation, and operation |

## C4ISR / C5ISR Products

### GATR Inflatable Satellite Communication Terminals

Cubic's GATR Technologies manufactures portable, inflatable SATCOM terminals providing high-bandwidth communications in tactical and austere environments. Key innovation: up to 85% reduction in logistical size/weight compared to rigid deployable antennas of equivalent aperture.

#### GATR 1.2m Antenna

| Parameter | Value |
|---|---|
| **Antenna Diameter** | 1.2 m |
| **Pack-Out** | Single airline-checkable case |
| **Weight** | < 75 lbs (< 34 kg) total |
| **Weight Reduction** | Up to 50% vs rigid deployable antennas of similar size |
| **Frequency Band** | Ku-band |
| **Key Advantage** | Full 1.2m dish provides narrower beamwidth, lower noise floor, better G/T than sub-meter dishes |
| **Environmental** | MIL-STD-810G tested |

#### GATR 2.4m Antenna

| Parameter | Value |
|---|---|
| **Antenna Diameter** | 2.4 m |
| **Pack-Out** | As few as 2 airline-checkable cases |
| **Weight Per Case** | < 100 lbs (< 45.4 kg) each |
| **Weight Reduction** | Up to 80% vs rigid deployable antennas |
| **Frequency Bands** | Ku-band, C-band, WGS-certified X-band, Ka-band |
| **Setup Time** | < 30 minutes to on-satellite |
| **Environmental** | MIL-STD-810G tested; durable in extreme temperatures; stable in high winds |
| **Technology** | Inflatable radome with flexible parabolic reflector mounted at equator |

#### GATR 4.0m Antenna

| Parameter | Value |
|---|---|
| **Antenna Diameter** | 4.0 m |
| **Pack-Out** | As few as 4 cases |
| **Total Weight** | < 400 lbs (< 180 kg) |
| **Weight Reduction** | > 80% vs rigid deployable hubs |
| **Frequency Band** | Ku-band |

## Counter-UAS Relevance

Cubic Defence Australia does not publicly market a dedicated counter-UAS hardware product. However, their capabilities are relevant to the counter-UAS ecosystem:

| Capability | Counter-UAS Application |
|---|---|
| **LVC Training with Virtual UAVs** | Training for C-UAS scenarios using simulated drone threats |
| **C4ISR Networking (DTECH M3)** | Secure tactical networking for C-UAS command and control |
| **GATR SATCOM** | Deployable communications for C-UAS coordination in austere environments |
| **CATS Metrix / SPEAR** | Exercise control and AAR for C-UAS training exercises |
| **Geometric Pairing** | Next-generation weapon effect simulation applicable to C-UAS engagement training |
