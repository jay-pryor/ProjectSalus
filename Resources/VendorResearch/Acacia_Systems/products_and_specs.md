# Acacia Systems - Products and Specifications

**Company:** Acacia Systems Pty Ltd, Hendon, South Australia
**Founded:** 1992 (originally as Acacia Research, by Ted Huber)
**Role in LAND 156:** Key partner under LOE1 providing the sovereign Cortex C2 command and control system
**Website:** [acaciasystems.com.au](https://www.acaciasystems.com.au)

Acacia is a leading Australian defence software and systems engineering company developing advanced mission management systems and tactical support applications for the Maritime (surface and subsurface), Land, and Air environments. The company maintains significant depth of skills in data fusion, tracking, sensor performance, data analysis, and environmental mathematical modelling.

---

## 1. Cortex C-UAS C2 - Command and Control System

Cortex is the central command and control platform for the LAND 156 counter-small UAS capability. It is an integrated multi-sensor, multi-effector counter-UAS command and control system.

### Capabilities

| Parameter | Detail |
|---|---|
| **System Type** | Integrated multi-sensor, multi-effector C-UAS C2 platform |
| **Architecture** | Open architecture, open-standard interfaces |
| **Sensor Integration** | Rapid sensor and effector configuration and integration |
| **Data Fusion** | Fuses data from multiple sensors into a coherent threat picture |
| **Engagement Management** | Manages engagement workflows under human-in-the-loop command framework |
| **Compute Efficiency** | Delivers comparable capability at 25% of the compute power of comparable systems |
| **Platform Weight/Volume** | Comparable advantages (reduced) over competing systems |
| **Robustness** | Low-level distributed processing and data persistence; no single point of failure causes outage |
| **UxV Support** | Manages, controls, and displays data from multiple sensors and systems, including unmanned vehicles |
| **Scalability** | Modular and scalable; supports individual deployments to large-scale operations |
| **Deployment Flexibility** | Supports a variety of deployments across Defence |

### Design Philosophy

- Designed alongside the warfighter with significant input from sovereign Australian sensor and effector vendors
- Supports rapid integration through open-standard interfaces
- Evolved to counter the changing threat landscape
- Lightweight platform delivering heavyweight mission system capability across multiple operational domains

### LAND 156 Integration

Under LAND 156 LOE1, Cortex integrates the following systems:
- **Sensors:** Department 13 (passive RF), Echodyne MESA radars, EO/IR systems
- **Effectors:** EOS Defence Systems (Slinger kinetic, directed energy)
- **Additional Effectors:** L3 Harris VAMPIRE laser-guided rocket system
- **C2 Framework:** Human-in-the-loop engagement authority

Leidos (systems integrator) partnered with Acacia Systems to deliver an initial CsUAS capability demonstrated at Southern Arrow 25 in December 2025.

### Cortex Platform Variants

Cortex has been customised for multiple domains:
- **Cortex C-UAS C2:** Counter-UAS command and control (LAND 156)
- **Cortex Mission Management System:** Deployed on Bombardier Challenger 604 aircraft for the Australian Maritime Safety Authority

## 2. Onyx - Intelligent Tracking System

Onyx is an advanced auto target detect, track, and localisation engine using multi-sensor data fusion.

### Capabilities

| Parameter | Detail |
|---|---|
| **Function** | Auto target detect, track, and localisation engine |
| **Output** | Full-state solutions: range, bearing, course, and speed |
| **Data Fusion** | Fuses data from multiple sensors and/or multiple platforms |
| **Detection Performance** | Increased probability and range of detection over baseline sensors |
| **Tracking** | More accurate tracking of targets |
| **False Alarm Rate** | Greatly reduced false alarm rate through advanced algorithms |
| **Domain Origin** | Initially developed for anti-submarine warfare (ASW) |

### Onyx Force (Multi-Platform Extension)

| Parameter | Detail |
|---|---|
| **Function** | Multi-platform cooperative data fusion |
| **Architecture** | Architectural fusion node framework |
| **Integration** | Combines raw pre-track sensor data across cooperating ASW assets |
| **Benefits** | Increased detection range, reduced time to solution, increased overall defence capability of the ASW screen |
| **Sensor Domains** | Active and passive sonar |

### Deployments

- Integrated Sonar System aboard Australia's Hobart-class Air Warfare Destroyers
- Multi-sensor data fusion for RAN surface fleet anti-submarine capabilities
- Collins-class submarine systems (via Reflex integration)

---

## 3. Reflex - Command Decision Analytics

Reflex is a data management and replay system for mission analysis and tactical development.

### Capabilities

| Parameter | Detail |
|---|---|
| **Function** | Command decision analytics tool |
| **Data Capture** | Captures, records, and references all data geospatially and temporally |
| **Replay** | Instantaneous or post-mission replay |
| **Analysis** | Reconstructs and analyses events and incidents for tactical development |
| **Origin** | Developed for Collins-class submarine mission data management |

### Deployments

- Collins-class submarines (upgraded version redeployed)
- Allied Navy shore-based submarine combat system simulator (pending sea trial)
