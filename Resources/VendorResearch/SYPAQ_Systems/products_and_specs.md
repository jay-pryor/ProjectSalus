# SYPAQ Systems - Products and Specifications

## Company Overview

SYPAQ Systems Pty Ltd is a wholly Australian, veteran-owned and operated company founded in 1992. Based in Melbourne, Victoria, SYPAQ is a consulting, systems integration, and support company serving all branches of Australian Defence, Aerospace, and Homeland Security markets. SYPAQ provides sovereign Australian defence solutions with a focus on autonomous systems, ISR, and advanced payloads.

SYPAQ has been listed on Australia's counter-sUAS advisory panel, established to advise Defence and other Australian Government agencies on counter small uncrewed aerial systems (CsUAS) services.

---

## Corvo Family of Autonomous Systems

SYPAQ's unmanned systems brand, **Corvo**, encompasses a range of next-generation unmanned aerial systems (UAS), unmanned surface vessels (USV), and unmanned ground vehicles (UGV).

---

### Corvo PPDS (Precision Payload Delivery System)

The "cardboard drone" -- a disposable, flatpack delivery UAS made of waxed foamcore (foamboard).

| Parameter | Value |
|---|---|
| **Empty Weight** | 2.4 kg (5.3 lb) |
| **Maximum Payload** | 3 kg (6.6 lb) internally |
| **Range** | Up to 120 km (75 mi) |
| **Propulsion** | Electric motor, lithium-ion battery |
| **Endurance** | Sufficient for 120 km range |
| **Launch Method** | Catapult (lightweight frame with integrated launch rails and spring-loaded spool) |
| **Unit Cost** | AUD $1,000 - $5,000 (approx. USD $670 - $3,350) |
| **Construction** | Waxed foamcore (foamboard), self-assembly flatpack |
| **Control System** | Tablet-PC control centre (COTS hardware, FOSS software, military grade) |
| **Navigation** | GPS-independent capable (can operate in GPS-denied / RF-jammed environments) |
| **Stealth Features** | Small size, foamboard construction, no communications link in flight |
| **Security** | Encrypted flightpath data; command controller location cannot be identified from captured unit |

**Combat Use**: More than 500 PPDS platforms delivered to Ukraine; customised for reconnaissance and attack roles in the Russo-Ukrainian War.

### Corvo PPDS-HL (Heavy Lift)

A larger variant of the base PPDS optimised for multi-role operations.

| Parameter | Value |
|---|---|
| **Range (6 kg payload)** | 80 km |
| **Range (3 kg payload)** | Up to 200 km |
| **Payload Capacity** | Up to 6 kg |
| **Payload Mounting** | Internal fuselage bay + external wing hardpoints |
| **External Stores** | Hardpoints enable in-flight payload drop |
| **ISR Capability** | Can carry lightweight ISR payloads simultaneously with external stores |
| **Multi-Role** | Payload delivery, ISR, and strike in a single sortie |

### CorvoX (Next-Generation SUAS)

A military-grade small UAS featuring thrust-vectored VTOL, under contract for the Australian Army's DEF129 programme. Developed with Teledyne FLIR as an EO/IR payload partner.

| Parameter | Value |
|---|---|
| **Weight** | < 2 kg (< 4.4 lb) |
| **Wingspan** | 82 cm (32.3 in) |
| **Maximum Speed** | 100 km/h (62 mph) |
| **Endurance** | Up to 50 minutes |
| **Operational Range (est.)** | 120 - 150 km (payload dependent) |
| **Wind Resistance** | Up to 55 km/h (35 mph) |
| **Launch Method** | Hand-thrown or ground VTOL (thrust-vectored) |
| **Configuration** | Thrust-vectored VTOL fixed-wing hybrid |
| **Australian Content** | > 90% |

#### CorvoX Payload Options

| Payload | Details |
|---|---|
| **Corvo Guarda 70-DN EO/IR** | 640 x 512 resolution Teledyne FLIR Boson thermal sensor |
| **ISR Mode** | 1-2 hours endurance with ISR payload |
| **Loitering Munition Mode** | Optional warhead module for kinetic effects (sub-5 kg class); reduced endurance |

#### CorvoX Key Features

- Novel memory structure prevents intelligence leakage during missions
- Onboard AI capabilities with continuous updates
- Mesh networking capabilities
- Man-packable system reducing physical and cognitive operator load
- Export-ready expected 2026
- Delivery to Australian Army under DEF129 programme commenced December 2025

## ISR Integration and Support

SYPAQ provides integration and support of COTS/MOTS ISR products including:

- UAVs and UAS platforms
- EO/IR sensors and gimbals
- Processing, Exploitation, and Dissemination (PED) hardware and software
- ISREW sensor systems
- Data fusion systems
- Tactical communications technologies
