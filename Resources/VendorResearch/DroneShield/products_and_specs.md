## Dismounted / Handheld Products

### DroneGun Mk4

**Category:** Handheld C-UAS effector (pistol-form factor)
**Released:** April 2023

The DroneGun Mk4 is a rugged, compact handheld counter-drone jammer designed for single-hand operation. It simultaneously disrupts control, navigation, and video feeds of multiple drones.

| Specification | Details |
|---|---|
| **Weight** | 3.2 kg (7.05 lbs) including battery |
| **Dimensions** | 660 x 356 x 213 mm (26" x 14" x 8.4") |
| **Effective Range** | Not publicly disclosed (classified; described as enhanced over Mk3) |
| **Effective Frequencies** | Wide range of ISM bands + GNSS frequency bands |
| **IP Rating** | IP67 |
| **Operating Temperature** | -20 C to +55 C (-4 F to +131 F) |
| **Battery** | Quick-release rechargeable lithium-ion (NATO-standard) |
| **Battery Life** | 1+ hour continuous operation |
| **Startup Time** | Less than 3 seconds |
| **Features** | Picatinny rails (top/bottom), QD sling points, improved pistol grip, safety selector switch |
| **Construction** | Ruggedized; IP67 certified for harsh environments |

**Notes:**
- Effective against commercial and military UAS using COTS communications (e.g., Russian Orlan-10).
- Non-kinetic jamming only; no protocol manipulation or cyber tactics.
- Not FCC authorized; U.S. sales restricted to government agencies.

## Radar Integration

DroneShield does not manufacture its own radar but integrates third-party radar systems into the DroneSentry architecture. The company's product page lists radar as a configurable sensor option for the DroneSentry fixed-site system.

**Integration approach:**
- DroneSentry-C2 and SensorFusionAI fuse radar data with RF, optical, and ADS-B data.
- Open architecture and common APIs allow integration of various radar manufacturers.
- Radar data is cross-referenced with RF detections to reduce false alarms and improve tracking accuracy.

Third-party radar partners are not publicly enumerated but the architecture supports any radar providing standard data outputs.
