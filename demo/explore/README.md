# Exploration Scenarios

Scenarios in this directory are designed to stress specific parts of the Salus
pipeline and surface issues for the Iterative Improvement phase.

## Naming convention

```
ENN_short_description/
```

- `NN` — zero-padded sequence number (E01, E02, …)
- `short_description` — snake_case description of the scenario's focus

## Contents of each scenario directory

Each directory typically contains:
- `scenario.yaml` — the Salus scenario config
- `generate.py` — script to run the scenario and open the viewer (optional)
- `notes.md` — what was being tested and what was found (added after the run)
